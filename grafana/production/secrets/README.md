# Grafana Production Secrets

Create the following files with real secrets (do not commit them):

- `grafana_admin_password.txt`
- `grafana_db_password.txt`
- `navi_db_password.txt`

Examples are provided with `.example` suffix. Copy and replace contents:

```bash
cp grafana_admin_password.txt.example grafana_admin_password.txt
cp grafana_db_password.txt.example grafana_db_password.txt
cp navi_db_password.txt.example navi_db_password.txt
```
